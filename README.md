# CryptoScoreBoard

## Local setup with docker
Start docker containers
```docker-compose up -d```

### Local Db access :
- Host : db
- user : root
- password : root
- database : test

### Browser access :
Default : localhost port 80
Recomanded : add cryptoscoreboard.local to /etc/hosts

### phpmyadmin
http://localhost:8080
See Db access for login informations
